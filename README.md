# ✂️ 가위(GAWI) : 원고 슬라이서

웹툰 원고를 플랫폼 규격에 맞게 자동으로 슬라이스하는 무료 웹 도구입니다.

## 기능

- ZIP 또는 이미지(JPG/PNG) 업로드
- 플랫폼별 원고 규격에 맞게 자동 슬라이스
- ZIP 여러 개 일괄 처리 지원
- 다운로드 시 파일명 지정 가능
- 서버 업로드 없이 브라우저에서 처리 (원고 유출 걱정 없음)

## 지원 플랫폼

| 플랫폼 | 규격 |
|--------|------|
| 카카오페이지 · 리디 · 미스터블루 · 무툰/큐툰 · 코미코 · 올툰 | 760 × 990 px |
| 네이버웹툰 (요일 / 매일+) | 760 × 3,000 px |
| 기타 플랫폼 | 준비 중 🚧 |

## 사용법

1. 사이트 접속
2. ZIP 또는 이미지 파일 업로드
3. 플랫폼 버튼 클릭
4. ZIP 또는 개별 파일로 다운로드

## 🇬🇧 English

### GAWI : Webtoon Manuscript Slicer

A free web tool that automatically slices webtoon manuscripts to fit platform specifications. No server upload needed — everything runs in your browser, keeping your artwork safe and private.

**Features:** Upload ZIP or image files (JPG/PNG) → Select platform preset → Auto-slice to exact pixel dimensions → Download as ZIP or individual files. Supports batch processing for multiple ZIP files with custom file naming.

**Supported platforms:** KakaoPage, Ridi, Mr.Blue, Mootoon/Cutoon, Comico, Alltoon (760×990px), Naver Webtoon (760×3000px). More platforms coming soon.

**Keywords:** webtoon image slicer, webtoon manuscript cutter, manga panel splitter, comic image resizer, webtoon upload tool, vertical scroll comic slicer, webtoon auto crop, webcomic image divider
